⚙️ Supabase einrichten

Einmalige Konfiguration — danach ist die App per Link für alle Kollegen zugänglich.

3 Schritte in Supabase:

1
Gehen Sie zu supabase.com → Kostenlos registrieren → Neues Projekt anlegen
2
Im Projekt: SQL Editor öffnen und diesen Befehl ausführen:
CREATE TABLE protokolle (id text PRIMARY KEY, data jsonb, updated_at timestamptz DEFAULT now());
CREATE TABLE ordner (id text PRIMARY KEY, name text, farbe text, pos integer DEFAULT 0);
3
Unter Project Settings → API: Project URL und anon public Key kopieren und unten einfügen
Aus: Project Settings → API → Project URL
Aus: Project Settings → API → anon public
